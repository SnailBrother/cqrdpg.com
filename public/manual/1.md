# 💪 俯卧撑完全指南

## 📌 简介

俯卧撑是最经典的自重训练动作之一，主要锻炼胸大肌、三角肌前束和肱三头肌，同时能强化核心稳定性。无需器械，随时随地都可以练习，是提升上肢力量的黄金动作。

![俯卧撑示意图](https://images.unsplash.com/photo-1571019613454-1cb2f99b2d8b?w=600&h=400&fit=crop)

## 🎯 目标肌群

| 主要肌群 | 次要肌群 | 稳定肌群 |
|---------|---------|---------|
| 胸大肌 | 三角肌前束 | 腹直肌 |
| 肱三头肌 | 前锯肌 | 竖脊肌 |
| | 喙肱肌 | 臀大肌 |

## 📝 标准动作步骤

### 准备姿势
1. **双手位置**：手掌撑地，略宽于肩，手指朝前
2. **身体姿态**：从头到脚呈一条直线，不要塌腰或撅臀
3. **双脚并拢**：脚尖着地，双脚可以并拢或稍分开
4. **核心收紧**：腹部收紧，想象肚脐贴向脊柱

### 动作执行
5. **下降阶段**：吸气，弯曲手肘，身体缓慢下降
6. **最低点**：胸部距离地面约2-3厘米，手肘呈90度
7. **推起阶段**：呼气，用力推起身体回到起始位置
8. **顶峰收缩**：在顶部稍作停顿，感受胸肌收缩
